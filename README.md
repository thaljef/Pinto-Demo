Pinto-Demo
==========

Tools and data for my live Pinto demonstrations


Prerequisites
=============


* Install a perl

* Install cpanm

* Install pinto

```bash
curl -L http://getpinto.stratopan.com | bash
```

* Install cleo

```bash
cpanm App::Cleo
```

Running The Demo
================

```bash
git clone git@github.com:thaljef/Pinto-Demo.git
cd Pinto-Demo
cleo demo.commands
```
